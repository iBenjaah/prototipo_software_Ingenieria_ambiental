# Versión para Presentación

## Ejecutable con Datos Pre-cargados

Para facilitar las demostraciones y presentaciones en la universidad, se ha creado una versión especial del ejecutable que incluye datos de ejemplo pre-cargados.

## Archivo

**`dist\SistemaGestionAmbiental_Presentacion.exe`**

## Características

✅ **Sin consola visible** - Ejecuta en segundo plano  
✅ **Abre navegador automáticamente** - No necesitas escribir la URL  
✅ **Datos pre-cargados** - Incluye datos de ejemplo automáticamente  
✅ **Listo para presentar** - Solo doble clic y funciona  

## Cómo Usar

1. **Doble clic** en `SistemaGestionAmbiental_Presentacion.exe`
2. **Espera 2-3 segundos** - El sistema carga los datos y abre el navegador
3. **¡Listo para presentar!** - Todos los módulos tienen datos de ejemplo

## Datos Incluidos

El ejecutable incluye automáticamente:
- **261 registros** de producción (últimos 12 meses)
- **1,132 registros** de consumos (agua, energía, zinc, KOH)
- **168 registros** de residuos (pilas, efluentes)
- **124 registros** de reciclaje (zinc, acero, KOH recuperados)

## Ventajas para Presentación

- ✅ No necesitas generar datos manualmente
- ✅ El sistema se ve completo desde el primer momento
- ✅ Todos los gráficos y tablas tienen información
- ✅ Los indicadores se calculan correctamente
- ✅ Ideal para demostraciones en vivo

## Notas

- Los datos se cargan automáticamente la primera vez que se ejecuta
- Si la base de datos ya tiene datos, no los sobrescribe
- Los datos cubren los últimos 12 meses de operación ficticia

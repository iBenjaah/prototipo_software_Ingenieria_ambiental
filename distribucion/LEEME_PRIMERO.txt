========================================
SISTEMA DE GESTIÓN AMBIENTAL
Baterías Zinc-Aire
========================================

BIENVENIDO

Este paquete contiene el Sistema Informático de Gestión Ambiental
para el seguimiento de variables críticas en la producción y ciclo
de vida de baterías zinc-aire tipo botón.

========================================
OPCIONES DE INSTALACIÓN
========================================

OPCIÓN 1: USAR EJECUTABLE (MÁS FÁCIL)
--------------------------------------
1. Doble clic en: SistemaGestionAmbiental.exe
2. Abrir navegador en: http://127.0.0.1:5000
3. ¡Listo! No requiere instalación adicional.

OPCIÓN 2: INSTALACIÓN MANUAL
-----------------------------
1. Ejecutar: instalar.bat
2. Ejecutar: iniciar.bat
3. Abrir navegador en: http://127.0.0.1:5000

OPCIÓN 3: GENERAR DATOS DE EJEMPLO
-----------------------------------
Para ver el sistema con datos ficticios:
- Ejecutar: generar_datos.bat
